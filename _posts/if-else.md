### Ruby

```ruby

    puts 'yes'
else
    puts 'no'
end
```

### Python

```python
if True:
    print('yes')
elif False:
    print('no')
else:
    print('n/a')
```

### Javascript

```javascript
if (true) {
  console.log("yes");
} else {
  console.log("no");
}
```

